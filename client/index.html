Chat Embedding Methods Demo

This page demonstrates 5 different ways to embed the same Socket.IO chat service.

(A) iframe Embed

<iframe src="http://localhost:5000/chat-ui?source=iframe"></iframe>

(B) Script-based Widget

<script src="http://localhost:5000/widget.js"></script>

Check the bottom-right corner of the page!

(C) JavaScript SDK

ChatApp.init({ websiteId: "demo" });
ChatApp.open();

SDK chat will appear floating on the bottom-left.

(D) Web Component

<chat-widget></chat-widget>

(E) Direct DOM Injection

initDomChat('dom-target');

This one is styled differently (Dark Mode) to show it's internal to the host page.
